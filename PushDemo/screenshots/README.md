# Screenshots for Practical Report

This directory contains screenshots referenced in the PRACTICAL_4_REPORT.md file.

## Required Screenshots

### Step 10: QR Code Scanning
**Filename:** `step10_qr_code.png`

**Description:** Screenshot showing the QR code being scanned with Expo Go app or Camera app

**What to capture:**
- The terminal window displaying the QR code
- The Expo Go app scanning the QR code
- Or the iOS Camera app with the Expo notification to open in Expo Go

---

### Step 11: Push Token Display
**Filename:** `step11_push_token.png`

**Description:** Screenshot showing the push token displayed on the device screen

**What to capture:**
- The app running on your device
- The push token clearly visible (format: ExponentPushToken[...])
- The full UI showing "Expo Push Demo" title and token text

---

## How to Add Screenshots

1. Take screenshots on your device during Steps 10 and 11
2. Transfer the images to this `screenshots/` folder
3. Rename them to match the filenames above:
   - `step10_qr_code.png`
   - `step11_push_token.png`
4. The report will automatically display these images

## Image Guidelines

- **Format:** PNG or JPG
- **Resolution:** High enough to read text clearly
- **Orientation:** Portrait for device screenshots, landscape for terminal screenshots
- **Size:** Keep under 2MB per image for easy sharing
